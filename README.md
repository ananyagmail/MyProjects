# Cat vs Dog Image Classifier

A Machine Learning desktop application that classifies uploaded images as either a Cat or a Dog.

## Features

- Upload any image
- Predict Cat or Dog
- Desktop GUI using Tkinter
- TensorFlow/Keras model

## Tech Stack

- Python
- TensorFlow
- Keras
- Tkinter
- OpenCV
- Teachable Machine

## How to Run

```bash
pip install -r requirements.txt
python gui.py
